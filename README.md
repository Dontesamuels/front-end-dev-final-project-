Dev Project Tracker is a React Single Page Application (SPA) built with Vite that helps beginner and aspiring developers organize, manage, and track their coding projects in one centralized dashboard. The application demonstrates core frontend engineering skills including component architecture, multi-route navigation using React Router, global state management with the React Context API, and local state handling with useState. Users can create projects, add and toggle tasks, write notes, and monitor progress, while also exploring developer resources fetched from the public GitHub REST API with proper loading and error handling. The application follows a mobile-first responsive design using CSS Grid and Flexbox, includes basic unit testing with Vitest and React Testing Library, and is deployed on Vercel with version control managed through Git and GitHub. This MVP serves as the foundation for a future full-stack version that will include authentication, persistent database storage, and enhanced collaboration features.
